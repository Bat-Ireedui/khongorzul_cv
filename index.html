Товч танилцуулга

Банкны мэдээллийн технологийн салбарт банкны суурь систем, олон улсын төлбөр тооцооны систем болох SWIFT болон FCM зэрэг mission-critical financial messaging системүүд дээр 5 жилийн туршлагатай. Одоогоор Худалдаа, хөгжлийн банкинд ахлах мэргэжилтнээр ажиллаж, 5–6 мэргэжилтний бүрэлдэхүүнтэй багийн ажлыг төлөвлөн удирдаж, Linux (RHEL), Oracle, OpenShift орчинд суурилсан дэд бүтэц, production системийн тогтвортой ажиллагаа, high availability болон disaster recovery шийдлүүдийг хариуцан ажиллаж байна.

SWIFT ISO 20022 шилжилт, core banking system upgrade зэрэг томоохон төслүүдийг удирдан хэрэгжүүлж, system integration, migration болон deployment үйл ажиллагааг амжилттай гүйцэтгэсэн. Incident, problem болон change management процессыг удирдан root cause analysis хийх замаар системийн найдвартай байдлыг сайжруулахад хувь нэмэр оруулсан.

Linux, Oracle Database, Red Hat OpenShift болон GitLab CI/CD орчинд ажиллах практик туршлагатай бөгөөд enterprise түвшний системүүдийн өндөр хүртээмж, тасралтгүй ажиллагаа болон continuous improvement-ийг ханган байгууллагын бизнесийн үйл ажиллагааг дэмжин ажилладаг.
