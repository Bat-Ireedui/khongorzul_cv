I

Бүлэг I

Товч танилцуулга

Банкны мэдээллийн технологийн салбарт банкны суурь систем, олон улсын төлбөр тооцооны SWIFT болон FCM зэрэг mission-critical financial messaging системүүд дээр таван жилийн туршлага хуримтлуулсан. Одоо Худалдаа, хөгжлийн банкинд ахлах мэргэжилтнээр ажиллаж, 5–6 мэргэжилтний бүрэлдэхүүнтэй багийн ажлыг төлөвлөн удирдаж, Linux (RHEL), Oracle, OpenShift орчинд суурилсан дэд бүтэц, production системийн тогтвортой ажиллагаа, high availability болон disaster recovery шийдлүүдийг хариуцан ажиллаж байна.

SWIFT ISO 20022 шилжилт, core banking system upgrade зэрэг томоохон төслүүдийг удирдан хэрэгжүүлж, system integration, migration болон deployment үйл ажиллагааг амжилттай гүйцэтгэсэн. Incident, problem болон change management процессыг удирдан root cause analysis хийх замаар системийн найдвартай байдлыг сайжруулахад хувь нэмэр оруулсан. Enterprise түвшний системүүдийн өндөр хүртээмж, тасралтгүй ажиллагаа болон continuous improvement-ийг ханган байгууллагын бизнесийн үйл ажиллагааг дэмжин ажилладаг.

2023.10 — ОДООГ ХҮРТЭЛ

Ахлах мэргэжилтэн

Худалдаа, хөгжлийн банк (TDBM)

Банкны суурь систем, SWIFT, FCM болон бизнесийн критик системүүдийн найдвартай, тасралтгүй ажиллагааг хариуцахын зэрэгцээ 5–6 мэргэжилтний бүрэлдэхүүнтэй багийг удирдан, системийн шинэчлэл, интеграц болон томоохон төслүүдийг төлөвлөн хэрэгжүүлж байна.

Манлайлал, удирдлага

  • 5–6 мэргэжилтний бүрэлдэхүүнтэй багийн ажлыг төлөвлөх, хуваарилах, гүйцэтгэлд хяналт тавьж, мэргэжлийн удирдлагаар хангах
  • SWIFT ISO 20022 шилжилт, Core Banking System upgrade зэрэг томоохон төслүүдийг удирдан зохион байгуулах
  • Системийн өөрчлөлт, production deployment-тэй холбоотой техникийн шийдвэр гаргах, эрсдэлийг үнэлэх

Үндсэн чиг үүрэг

  • SWIFT систем болон холбогдох дэд бүтцийн availability, performance, security-г хангах
  • Core banking болон enterprise системийн upgrade, migration болон production go-live ажиллагааг удирдан гүйцэтгэх
  • Linux, Oracle, OpenShift орчны системийн ашиглалт, хяналт, засвар үйлчилгээ хийх
  • GitLab CI/CD ашиглан deployment, release болон автоматжуулалтын үйл ажиллагааг удирдах
  • Incident / Problem / Change management процессыг хэрэгжүүлж root cause analysis хийх
  • High Availability болон Disaster Recovery шийдлүүдийн тасралтгүй ажиллагааг хангах
  • Дотоод нэгж болон vendor байгууллагуудтай хамтран системийн уялдаа холбоог хангах
Гол төслүүд
  • SWIFT ISO 20022 Migration — Messaging system readiness, deployment & integration support
  • Core Banking System Upgrade & Migration — Environment setup, integration testing, production go-live support
  • FCM Messaging System Support — Financial messaging flow monitoring, troubleshooting & integration support

Онцлох үр дүн

  • Mission-critical banking systems (Core Banking, SWIFT, FCM)-ийн өндөр availability-г тогтвортой хангасан
  • Production incident-үүдийг шуурхай шийдвэрлэж, root cause analysis хийх замаар дахин давтагдах эрсдэлийг бууруулсан
  • System outage үед troubleshooting, escalation болон vendor coordination-ийг үр дүнтэй удирдсан
  • High availability болон disaster recovery орчны тогтвортой ажиллагааг хангасан
  • Cross-functional багууд болон бизнес нэгжүүдтэй үр дүнтэй хамтран ажилласан
Банкны систем хэдхэн минут зогсоход л мэдрэгддэг. Тасралтгүй ажиллагаа бол миний өдөр бүрийн хариуцлага.
